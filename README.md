# 仮名 · Kana Trainer

Treino direto de **hiragana** e **katakana**. Você vê o kana, digita o romaji, repete.

Sem login. Sem anúncios. Sem distrações.

## Modos de treino

| Modo | Descrição |
|------|-----------|
| **Hiragana** | Escolha quais linhas treinar (あ、か、さ…) |
| **Katakana** | Escolha quais linhas treinar (ア、カ、サ…) |
| **Misto** | Todos os hiragana e katakana embaralhados |

## Como usar

1. Selecione os caracteres que quer praticar
2. Clique em **Iniciar treino**
3. Digite a romanização do kana exibido e pressione **Enter**
4. Erros ficam registrados no painel lateral para revisão

## Stack

Vanilla HTML + CSS + JS — sem frameworks, sem dependências, sem build.
